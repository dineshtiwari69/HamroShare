## Run Locally

Clone the project

```bash
  git clone https://github.com/dineshtiwari69/HamroShare
```

Go to the project directory

```bash
  cd HamroShare
```

Install dependencies

```bash
  yarn install
```
Note : Remove homepage section from package.json
Start the server

```bash
  yarn start
```
